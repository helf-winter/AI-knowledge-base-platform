# AI 企业知识库管理平台演示文档

本文档用于课堂验收或答辩演示，演示主线对应 [课题要求.md](../reference/课题要求.md) 中的知识生产、知识管理、知识消费和自动学习闭环。

## 1. 演示目标

通过一个企业知识库场景，展示系统可以完成：

- 将企业文档导入知识库。
- 对知识进行检索、查看、编辑、审核和归档。
- 通过 AI 助手消费知识；知识库不足时自动切换通用回答。
- 对通用补答结果进行用户确认，并一键扩充到相近知识库或新建知识库。
- 查看任务、问答记录、审计日志，形成可追踪闭环。

## 2. 演示前准备

1. 启动后端：

```bash
uvicorn app.main:app --reload
```

2. 启动前端：

```bash
cd frontend
npm run dev
```

3. 打开页面：

```text
http://127.0.0.1:3000
```

4. 登录账号：

```text
admin / 123456
```

## 3. 推荐演示路径

### 3.1 首页仪表盘

进入 `/`，先说明系统不是单点问答工具，而是一个企业知识工作台。

重点展示：

- 知识文档数量。
- 知识条目数量。
- 自动学习待处理缺口。
- 任务状态。
- 最近问答和最近文档。

对应课题要求：基础体验、知识生产概览、知识消费概览。

### 3.2 知识生产：文档导入

进入 `/documents`。

演示步骤：

1. 上传一个企业制度、VPN 指南、报销流程或 Markdown 文档。
2. 等待解析任务完成。
3. 在文档列表中查看文件类型、解析状态和更新时间。
4. 点击文档详情，查看解析后的知识片段。

对应课题要求：手动录入、文档导入、知识生产。

### 3.3 知识管理：检索与条目管理

仍在 `/documents`，输入关键词，例如：

```text
VPN 怎么连接
```

展示检索结果中的：

- 来源文件。
- 文件类型。
- 更新时间。
- 命中摘要。
- 相似度分数。

再进入 `/admin`，展示：

- 新建知识条目。
- 编辑标题、作者、知识类型和 ACL。
- 审核通过。
- 归档或删除。

对应课题要求：创建、编辑、删除、检索、状态管理。

### 3.4 知识消费：自动问答与知识扩充

进入 `/skills`。

演示步骤：

1. 查看 Skill 能力目录。
2. 选择一个知识条目作为范围。
3. 一键创建专家助手。
4. 打开右下角浮动问答助手。
5. 先提问一个知识库能回答的问题，展示知识库引用。
6. 再提问一个知识库没有覆盖的问题，例如“VPN 如何申请？”。
7. 系统会自动切换为通用回答，并在回答框底部询问“是否扩充该知识内容？”。
8. 点击“扩充”，系统会把这次回答整理为知识片段，优先追加到相近知识库；没有相近内容时创建新的知识库文档。

重点展示：

- 知识充分时，回答来自知识库上下文并返回引用来源。
- 知识不足时，不会只说“不知道”，而是自动切换通用回答。
- 用户确认后可以把通用回答沉淀为知识。
- 返回 Trace ID。
- 问答记录被写入系统。

对应课题要求：专家 Agent、Skill、知识消费、响应流畅、AI 自动提炼新知识。

### 3.5 自动学习闭环

进入 `/tasks`。

演示步骤：

1. 查看自动学习入口。
2. 查看待处理知识缺口和建议草稿。
3. 调整分析小时数和最低置信度。
4. 点击“触发学习”生成自动学习任务。
5. 在任务监控中查看任务状态。

可讲解逻辑：

- 低置信度回答、用户反馈和未命中问题会进入知识缺口池。
- 自动学习会把相似问题聚类，生成建议标题和内容方向。
- 审核后可以沉淀为正式知识条目。

对应课题要求：AI 自动提炼新知识、知识闭环、任务监控。

### 3.6 追踪与审计

进入 `/conversations` 和 `/audit`。

展示：

- 每次问答的 query、answer、confidence、source refs、trace id。
- 用户操作日志。
- 自动学习和反馈操作也可以被追踪。

对应课题要求：平台完整性、可解释、可追踪。

## 4. 答辩讲解词

可以按下面这段概括：

```text
这个项目实现了一个企业知识库管理平台。它不仅能上传文档并检索，还把知识条目管理、自动问答、问答引用、知识扩充、自动学习和审计追踪串成闭环。用户可以先上传企业文档，系统解析成知识片段；问答助手会优先基于知识库回答，知识不足时自动切换通用回答，并询问用户是否扩充知识。用户确认后，系统会把新答案沉淀到最相近知识库，或创建新的知识库文档。这样知识库不是静态资料库，而是可以随着使用持续演进的知识工作台。
```

## 5. 验收检查清单

- 首页仪表盘可打开，统计信息能正常展示。
- 文档上传、文档列表、知识检索可用。
- 知识条目可以创建、编辑、审核和归档。
- 浮动问答助手可以基于知识库回答问题，也能在知识不足时自动通用补答。
- 点击“扩充”后，通用补答可以沉淀为新的知识片段或新知识库文档。
- 问答记录能展示置信度、引用来源和 Trace ID。
- `/tasks` 可以展示自动学习入口和任务监控。
- README、启动文档和演示文档能说明项目如何启动和演示。
