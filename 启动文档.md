# 本地启动说明

本文档用于说明如何在本地启动 AI 知识库管理平台的完整 Demo。

## 1. 前置条件

请确保本机已安装：

- Python 3.10+
- Node.js 18+
- npm
- PostgreSQL（或项目配置的兼容数据库）

## 2. 后端启动

### 2.1 安装依赖

在项目根目录执行：

```bash
pip install -r requirements.txt
```

### 2.2 配置环境变量

在项目根目录创建 `.env` 文件，并配置至少以下内容：

```env
DATABASE_URL=postgresql+psycopg://用户名:密码@localhost:5432/数据库名
DEEPSEEK_API_KEY=你的DeepSeekKey
DEEPSEEK_BASE_URL=https://api.deepseek.com
DEEPSEEK_MODEL=deepseek-chat
DEEPSEEK_THINKING_ENABLED=true
```

如果暂时不接入 DeepSeek，可以先不填 `DEEPSEEK_API_KEY`，系统会自动回退到检索式回答。

### 2.3 执行数据库迁移

```bash
alembic upgrade head
```

### 2.4 可选初始化脚本

如果项目中有初始化脚本，可以再执行：

```bash
python scripts/init_db.py
```

### 2.5 启动后端服务

```bash
uvicorn app.main:app --reload
```

后端默认地址：

```text
http://127.0.0.1:8000
```

## 3. 前端启动

### 3.1 配置前端环境变量

进入 `frontend` 目录，创建 `.env.local`：

```env
NEXT_PUBLIC_API_BASE=http://127.0.0.1:8000
```

### 3.2 安装依赖

```bash
cd frontend
npm install
```

### 3.3 启动前端

```bash
npm run dev
```

前端默认地址：

```text
http://localhost:3000
```

## 4. 演示步骤

1. 打开前端页面并登录
2. 上传文档或查看已有知识条目
3. 进入专家 Agent 页面查看/创建助手
4. 使用通用问答进行提问
5. 查看系统推荐的 Agent
6. 在浮窗中选择 Agent 并再次提问
7. 查看问答记录、操作记录和审计日志
8. 触发自动学习任务，查看知识提炼闭环

## 5. 常见问题

### 5.1 前端提示 401
通常是因为未登录或 token 失效，请重新登录。

### 5.2 接口请求失败
请确认：

- 后端是否已启动
- `NEXT_PUBLIC_API_BASE` 是否正确
- 数据库迁移是否已完成

### 5.3 数据库字段不存在
请先执行：

```bash
alembic upgrade head
```

然后重启后端服务。
